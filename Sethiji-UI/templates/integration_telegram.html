How to Setup Your Telegram Bots
1
Create a Telegram Bot

Open Telegram and search for @BotFather. Start a chat and send the command /newbot.

2
Name Your Bot

Give your bot a name (e.g., "CustomerMaster" or "SupplyMaster") and a username that ends with "bot".

3
Copy the API Token

BotFather will provide an API token. Copy this token and paste it in the appropriate field above.

4
Connect the Bot

Click the "Connect Bot" button to integrate your Telegram bot with Sethji.ai.
